1. Getting Started

Welcome to the text tutorial section. Here you will learn the fundamentals of web development and game design.

Prerequisites

Before you begin, ensure you have a code editor installed. We recommend VS Code, Sublime Text, or Notepad++.

Click the links on the left sidebar to navigate between different lessons.

2. HTML Basics

HTML stands for HyperText Markup Language. It is the skeleton of your website.

The Boilerplate

Every HTML file starts with the following structure:

<!DOCTYPE html>
<html>
  <head>
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello World</h1>
  </body>
</html>

3. CSS Styling

CSS stands for Cascading Style Sheets. While HTML is the skeleton, CSS is the skin and clothing.

Colors and Fonts

You can change the color of text using the color property, and the background using background-color.

body {
    background-color: #f4f4f4;
    color: #333;
}

4. JavaScript Logic

JavaScript adds interactivity to your website. It allows you to create games, calculators, and dynamic content.

Variables

Variables are containers for storing data values.

let score = 0;
const maxScore = 100;

if (score < maxScore) {
    console.log("Keep playing!");
}
